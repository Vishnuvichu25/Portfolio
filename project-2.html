Rise Above

Description

"Developed VR Mount Climbing game as for our major project. As a group of 2. we made the game called RISE ABOVE. Implemented Mechanics and given a immersive roller coster ride."

Contributions

  • 1. Developed and implemented Grabbing mechanics.
  • 2. Used unity spline and its compontes to make rail tracks and animate the rail in the track.
  • 3. Resolved issues related to camera movements, snapping, and vr auto height adjustment
  • 4. Implemented In game stamina for each hands, and conditions. Also made interactable poke buttons for in game UI.

Project Details

  • PLATFORM: VR
  • ENGINE: Unity 3D
  • ROLE: Developer
  • TOOLS USED: UNITY ENGINE, XR_Interactable, unity Slpine, Autohand VR
